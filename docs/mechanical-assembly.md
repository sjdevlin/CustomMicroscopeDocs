## Procedure

1. Build base frame and verify levelness.
2. Install linear stages and check travel smoothness.
3. Install optical mounting surface and verify flatness.
4. Route cables with strain relief and full-travel slack.
5. Lock final fasteners to target torque.

## Verification

- Stage can traverse full travel without binding.
- Repeatability checks meet tolerance.
- No cable snagging at motion limits.
